## Установка и сборка (`install.sh`)

Скрипт `install.sh` предназначен для установки необходимых системных пакетов и последующей сборки основного проекта. 

Перед первым запуском необходимо предоставить скрипту права на исполнение:
```bash
chmod +x install.sh

```

### Окружение

Версии библиотек зафиксированы под рабочую конфигурацию:

* **ОС:** Ubuntu 24.04.4 LTS
* **Компилятор:** GCC 13.3.0
* **Базовые библиотеки:** Eigen 3.4.0 | OpenBLAS 0.3.26 | LAPACK 3.12 | OpenMPI 4.1.6
* **Вспомогательные пакеты:** ScaLAPACK 2.2.1 | METIS 5.1.0
* **Решалщики:** Intel oneAPI MKL 2026.0 | MUMPS 5.8.0 | SuperLU_MT 4.0.2

### Использование

Скрипт поддерживает как полную, так и модульную установку:

| Команда | Описание |
| --- | --- |
| `./install.sh` | **Полная установка:** последовательное выполнение всех этапов. |
| `./install.sh apt` | Установка системных пакетов. |
| `./install.sh mkl` | Конфигурация Intel oneAPI MKL 2026.0. |
| `./install.sh mumps` | Сборка MUMPS 5.8.0. |
| `./install.sh superlu` | Сборка SuperLU_MT 4.0.2. |
| `./install.sh build` | Сборка проекта. |
| `./install.sh check` | Диагностика окружения (ничего не устанавливает). |

## Запуск бенчмарков (`run_bench.sh`)

Скрипт `run_bench.sh` автоматизирует процесс тестирования решателей на разреженных матрицах.

### Использование

```bash
./run_bench.sh <superlu|pardiso|mumps> <путь/к/матрице.mtx> [повторы]

```

### Аргументы

* `<superlu\|pardiso\|mumps>` — выбор используемого пакета решателя.
* `<путь/к/матрице.mtx>` — путь к файлу разреженной матрицы в формате Matrix Market.
* `[повторы]` — *(опционально)* количество запусков для усреднения результатов бенчмарка.

### Пример команды

```bash
./run_bench.sh pardiso ./matrices/lhr04.mtx 5

```
